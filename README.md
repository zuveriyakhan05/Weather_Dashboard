# Weather_Dashboard
🌦️ Weather Analytics Dashboard An interactive Power BI dashboard visualizing historical weather data. Analyzes correlations between temperature, humidity, and visibility. Features Power Query transformations and DAX measures to track atmospheric trends and seasonal anomalies through dynamic filters.
Project Overview
This repository features a dynamic Power BI Dashboard designed to analyze and visualize historical weather patterns. The project transforms raw meteorological data into actionable insights, focusing on the relationships between temperature, humidity, wind speed, and visibility.

# Key Visualizations & Features

The dashboard provides a multi-faceted view of environmental conditions:

Atmospheric Trends: Analysis of pressure (Sum of Press_kPa) and temperature fluctuations over time.

Visibility Correlation: A detailed bar chart and distribution plot exploring how wind speed impacts visibility.

Humidity Metrics: A waterfall chart showing the relative humidity percentage across different periods.

Thermal Analysis: Comparison between ambient temperature and dew point to understand air saturation levels.

Interactive Slicers: Users can filter the entire report by Weather Condition, Month, and specific Date/Time ranges for granular analysis.

# Tech Stack
Tool: Power BI Desktop

Data Source: CSV/Excel (Historical Weather Dataset)

Data Transformation: Power Query (DAX for custom measures)

Visualization: Advanced charts including Waterfall, Scatter, and Distribution plots.

# Insights Derived

Yearly Comparisons: Visualizing data trends primarily centered around 2012 to identify seasonal anomalies.

Wind & Visibility: Identifying the specific wind speed thresholds where visibility significantly drops or improves.

Monthly Distribution: A dedicated sidebar providing quick-look sums of temperature for specific months like April, August, and December.

# How to Use:-

Clone this repository.

Open the .pbix file using Power BI Desktop.

Interact with the slicers at the top to filter data by specific weather events (e.g., Fog, Rain, Clear).
